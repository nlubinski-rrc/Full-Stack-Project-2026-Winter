## Overall idea
We want to create a movie review system similar to IMDB / Rotten tomatos / Good reads / letterboxd

## User stories
- As a user, i want to be able to look online for movie reviews to ensure i dont waste my time with bad movies.
- As a movie reviewer, i want to spread my thoughts on what i thought about a movie to improve overall movie development.
- As a user, i want to have a watch list so i dont forget all the movies i want to watch.
